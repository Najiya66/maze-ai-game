# 🧩 Maze AI Game

An interactive maze game where the player competes against an AI to reach the goal first.

---

## 🎮 Features

- 🟢 Player vs 🤖 AI gameplay  
- 🧠 AI uses pathfinding algorithm (BFS)  
- 🎲 Random maze generation every game  
- 🎚️ Difficulty levels:
  - Easy → Slow AI  
  - Medium → Normal AI  
  - Hard → Fast AI  
- ▶️ Start button to begin game  
- 🔄 Restart option  
- 🎨 Modern UI with animations

## 🚀 How to Run

1. Download or clone the repository  
2. Open `index.html` in browser  
3. Select difficulty level  
4. Click **Start Game**  
5. Use arrow keys to play  

---

## 🎯 Objective

Reach the goal before the AI!
